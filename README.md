# vue-router-transition

> vue router 切换动画

## 特性

* 模拟前进后退动画
* 基于css3流畅动画
* 基于sessionStorag，手动刷新不影响判断前进后退
* 返回可记录滚动条位置
* 前进后退的判断与路由路径规则无关
* 支持任何
